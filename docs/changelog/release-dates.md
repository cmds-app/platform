# Release dates

CMDS gets regular updates throughout the year. Each update first appears in our **Demo** environment — a safe space you can use to try things out — and then a week later it goes **Live** for everyone.

If you'd like to preview a new release before it goes Live, log in to Demo on or after the date listed below.
